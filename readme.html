Introduction

This plugin uses a combination of MySQL Full-Text Search and LIKE/REGEXP queries to find products, brands and categories in a fast way and sort them based on their relevance to the user query.

The results can be displayed immediately, in an autocomplete dropdown while the user is typing in a search box, and/or as a search results page with sortable product list and infinite scroll, once the user submits the search form (therefore as an alternative to the classic Zen Cart search results page).

Prerequisites

  • Zen Cart 1.5.7 or 1.5.8
  • PHP 7.4 through 8.2

Installation

Follow one of the following three chapters, depending on if you're currently using Instant Search v1, Instant Search v2, or you're installing Instant Search for the first time.

Upgrading from versions 1.x

  1. REMOVE all the plugin files from your site's filesystem before installing:
    • (up to version 1.0.2) searches.php
    • includes/templates/YOUR_TEMPLATE/jscript/jscript_instantSearch.js
    • includes/templates/YOUR_TEMPLATE/css/stylesheet_instantSearch.css
  2. Do a first-time install.

Upgrading from versions 2.x

Zen Cart 1.5.8

  1. REMOVE the following plugin files from your site's filesystem before installing:
    • includes/classes/ajax/zcAjaxInstantSearch.php
    • includes/languages/english/extra_definitions/YOUR_TEMPLATE/instant_search.php
    • includes/templates/YOUR_TEMPLATE/templates/tpl_ajax_instant_search_results.php
  2. Copy the plugin files to your site's filesystem.
    Rename YOUR_TEMPLATE to match the name of your site's template.
    DON'T rename the admin subdirectory under zc_plugins/InstantSearch/v3.0.0.
  3. If you're using the Bootstrap template on your store, copy the content of the bootstrap template files directory to your site's filesystem (overwrite files if asked).
  4. In the Admin, go to Modules > Plugin Manager, select Instant Search, click the Upgrade Available button and then Upgrade.
  5. In the same Admin page, click Clean Up to remove the previous plugin version files from the filesystem.
  6. In the file /includes/extra_configures/robots_pages_to_skip.php, add the line
    instant_search_result,
    under the line
    info_shopping_cart,

Zen Cart 1.5.7

  1. REMOVE the following plugin files from your site's filesystem before installing:
    • includes/classes/ajax/zcAjaxInstantSearch.php
    • includes/languages/english/extra_definitions/YOUR_TEMPLATE/instant_search.php
    • includes/templates/YOUR_TEMPLATE/templates/tpl_ajax_instant_search_results.php
  2. Write down your current Instant Search settings in the admin (Configuration > Instant Search), because they will be lost once you uninstall the current plugin.
  3. Uninstall the plugin via the Plugin Manager: in the Admin, go to Modules > Plugin Manager, select Instant Search and click the Uninstall button.
  4. Do a first-time install.
  5. Re-apply the previous Instant Search settings that you had written down on step 1.

First-time install

  1. Copy the plugin files to your site's file system.
    Rename YOUR_TEMPLATE to match the name of your site's template.
    DON'T rename the admin subdirectory under zc_plugins/InstantSearch/v3.0.0.
  2. If you're using the Bootstrap template on your store, copy the content of the bootstrap template files directory to your site's filesystem (overwrite files if asked).
  3. Install the plugin via the Plugin Manager: in the Admin, go to Modules > Plugin Manager, select Instant Search and click the Install button.
  4. Add the results listing page to the ROBOTS_PAGES_TO_SKIP value:
    • Zen Cart 1.5.8: in the file /includes/extra_configures/robots_pages_to_skip.php, add the line
      instant_search_result,
      under the line
      info_shopping_cart,
    • Zen Cart 1.5.7: in your template's version of the file /includes/languages/english/meta_tags.php, in the line that begins with define('ROBOTS_PAGES_TO_SKIP', add
      instant_search_result,
      right after
      info_shopping_cart,

Configuration

There are several configuration options available in the Admin (Configuration > Instant Search):

  • Dropdown - Enable
    When the user types into an input search box, display a dropdown with autocomplete search results.
  • Listing Page - Enable
    When the user submits a search form (advanced search form excluded), display the search results in a product listing page with infinite scroll.
  • Dropdown - Search Delay
    Delay the execution of the instant search by this time period (in milliseconds), after a character is entered, to prevent unnecessary queries to the database while the user is typing.
  • Dropdown - Maximum Number of Results
    Maximum number of results displayed in the dropdown.
  • Dropdown - Minimum Length of Search Query
    Minimum number of characters requested for the dropdown to be displayed.
  • Dropdown - Maximum Length of Search Query
    Maximum number of characters requested for the dropdown to be displayed.
  • Dropdown - Fields to Search and Order
    See Fields to search and order.
  • Dropdown - Include Related Products in the Results (Query Expansion)
    See Query expansion.
  • Dropdown - Display Images
    Display the product/category/manufacturer's image.
  • Dropdown - Display Product Price
    Display the product's price.
  • Dropdown - Display Product Model
    Display the product's model.
  • Dropdown - Display Category Count
    Display the number of products for the matched categories.
  • Dropdown - Display Manufacturer Count
    Display the number of products for the matched manufacturers.
  • Dropdown - Image Width
    Width of the product/category/manufacturer's image
  • Dropdown - Image Height
    Height of the product/category/manufacturer's image.
  • Dropdown - Highlight Search Terms in Bold
    Set to none for no highlight, query to highlight the user search terms, autocomplete to highlight the autocompleted text.
  • Dropdown - Input Box Selector
    CSS selector of the search input box(es). You might need to change it if you're using a custom template and the results dropdown is not showing.
  • Dropdown - Add Entry to Search Log
    Add the searched terms to the Search Log report (if Search Log plugin is installed).
  • Listing Page - Number of Results per Page
    Number of products per page displayed in the search results listing page
  • Listing Page - Fields to Search and Order
    See Fields to search and order.
  • Listing Page - Include Related Products in the Results (Query Expansion)
    See Query expansion.
  • Listing Page - Add Entry to Search Log
    Add the searched terms to the Search Log report (if Search Log plugin is installed).

Uninstall

  1. Uninstall the plugin via the Plugin Manager: in the Admin, go to Modules > Plugin Manager, select Instant Search and click the Uninstall button.
  2. Remove the plugin files from your site's filesystem.
